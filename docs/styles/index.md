# DSAV CSS

_Last updated: 2026-01-06_

DSAV CSS is a collection of utility classes and design tokens that form the foundation of the DSAV design system. It provides a consistent and flexible way to style web applications, ensuring a cohesive user experience across different components and layouts.

DSAV CSS represents the utility-first CSS framework that underpins the DSAV design system. It includes a variety of utility classes for layout, spacing, typography, colors, and more, allowing developers to rapidly build and customize user interfaces while adhering to the design principles of DSAV.
